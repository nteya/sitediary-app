# SiteDiary

SiteDiary is a construction site diary and reporting app built to support daily site supervision, structured reporting, and PDF-based record keeping.

## About
This project was created to help supervisors and site teams capture daily work information in a clear, organized, and professional way. It focuses on improving reporting quality, record keeping, and communication on site.

## Problem It Solves
On many construction and project sites, daily reporting can become inconsistent, rushed, or difficult to manage properly. Important information such as work completed, labor on site, delays, safety observations, and general site activities may not always be recorded in a structured way. This can make it harder to track progress, maintain accountability, and produce professional records when needed.

SiteDiary was built to make daily reporting more practical, structured, and reliable by giving supervisors and site teams a mobile-friendly way to capture information and generate clear PDF records.

## Why I Built This
I built SiteDiary because I am interested in creating practical software that solves real problems in working environments. I like building tools that improve the way people record information, communicate, and manage daily responsibilities.

Working around site-based environments made me realize that reporting is one of the most important parts of supervision and project control, but it is often still done in ways that are slow, inconsistent, or not well structured. I wanted to create an app that could simplify this process and make it easier for supervisors to produce organized daily records.

This project also challenged me technically because it involved thinking beyond just the user interface. I had to consider how information should be captured, structured, and turned into professional PDF reports that are useful in real working environments.

I built SiteDiary not only to test my software development skills, but also because I enjoy contributing to systems that help people work better, report better, and stay organized.

## Main Features
- Daily site diary entries
- Supervisor reporting workflow
- Workforce and task tracking
- Progress and site activity capture
- PDF report generation
- Structured daily records
- Site-based documentation support
