# SiteDiary

SiteDiary is a construction site diary and reporting app built to support daily site supervision, structured reporting, and PDF-based record keeping.

## About
This project was created to help supervisors and site teams capture daily work information in a clear and organized way. It focuses on improving reporting quality, record keeping, and communication on site.

## Main Features
- Daily site diary entries
- Supervisor reporting workflow
- Workforce and task tracking
- Progress and site activity capture
- PDF report generation
- Structured daily records
- Site-based documentation support

## Tech Stack
- React Native
- Expo
- Firebase
- Firestore
- JavaScript
